# 从策略迭代到价值迭代：研究者完整数学思考演进全过程（超详细溯源）

## 前言

在1960年霍华德提出**策略迭代**后，学界公认求解MDP必须严格遵守「两步分离范式」：

**完整策略评估（迭代收敛） → 策略提升（贪心更新策略）**

当时所有数学家、控制论学者都默认一条铁律：**必须先把当前策略的价值算到绝对精准（不动点），才能贪心更新策略**，否则价值存在误差，策略改进一定会失效、震荡、不收敛。

价值迭代的诞生，本质是**推翻了这条几十年的行业默认铁律**。

本文完整复刻：研究者如何发现漏洞、提出猜想、通过数学定理证明、最终合并两步、诞生价值迭代的完整思维链条。

## 第一阶段：1960年，学界固有认知（策略迭代的刻板框架）

### 1\.1 当时的标准算法：策略迭代

完整流程：

1. **初始化随机策略**：比如每个动作均等概率 `[0.5, 0.5]`

2. **策略评估（内层循环）**：固定策略，反复迭代贝尔曼期望方程，**直到整张价值表完全收敛、误差趋近于0**

3. **策略提升（外层单步）**：基于收敛后的精准价值，贪心选择最优动作，更新策略为 `[1, 0]`

4. 循环往复，直到策略不再变化

### 1\.2 当时学界的核心误区（关键卡点）

所有学者默认：

**只有 V 是当前策略的精准不动点，贪心出来的新策略才一定更优**。

如果 V 没收敛、存在误差，贪心得到的策略可能变差、算法震荡、无法收敛。

因此：**策略评估必须跑完无穷次迭代，绝对不能省略、不能合并**。

### 1\.3 策略迭代的致命缺陷（研究者的痛点）

每次只改动少量状态的策略，都要**全局全表重新收敛评估**，计算冗余爆炸、效率极低。

研究者开始思考一个问题：**能不能不跑完完整收敛，也能更新策略？**

## 第二阶段：第一个数学突破——策略改进定理（打破刻板认知）

### 2\.1 核心定理内容

对**任意一个价值函数 V（不管是否收敛、是否精准）**：

只要基于该 V 做贪心策略提升：$\pi'(s) = \arg\max_a Q(s,a,V)$

则一定满足：$V^{\pi'} \ge V^\pi$

**新策略的真实全局价值，一定大于等于旧策略**。

### 2\.2 颠覆式推论（关键思维跃迁）

原来根本不需要 V 收敛到不动点！

**随便一个粗糙的、未收敛的、有误差的价值表，贪心出来的策略依然不会变差**。

这直接推翻了1960年以来的核心误区：**完整收敛评估不是策略提升的前置必要条件**。

### 2\.2\.1 补充：策略改进定理 完整严格数学证明（核心溯源）

为解决研究者最大的疑虑：**凭什么未收敛、有误差的V，贪心出新策略也不会变差？**下面给出学界原始最简推导，无跳跃、全覆盖。

**已知前提**：

1\. 设任意旧策略 $\pi$，任意价值函数 $V$（无需是 $V^\pi$、无需收敛）；

2\. 贪心生成新策略：$\pi'(s) = \arg\max_a Q(s,a,V)$；

3\. 动作价值定义：$Q(s,a,V) = \mathbb{E}[r+\gamma V(s')|s,a]$。

**第一步：贪心策略的核心不等式（最关键一步）**

因为 $\pi'$ 是对每个状态 $s$ 取Q值最大的动作，所以天然满足：

$\max_a Q(s,a,V) \ge \sum_a \pi(a|s)Q(s,a,V)$

左侧是新贪心策略的单动作最优Q值，右侧是旧策略的动作加权平均Q值。

通俗解释：**单个最优选项的收益，一定大于等于所有选项的平均收益**，这是不等式天然成立的基础。

代入新旧策略可得：

$\mathbb{E}_{a\sim\pi'}[Q(s,a,V)] \ge \mathbb{E}_{a\sim\pi}[Q(s,a,V)]$

进一步简化为核心关系式：

$T^{\pi'}V \ge T^\pi V$

该式**不要求V是任何策略的真实价值**，对任意V恒成立。

**第二步：迭代递推证明策略单调提升**

对任意状态 $s$，我们有：

$V(s) \le T^{\pi'}V(s)$

该式含义：用新策略对应的期望算子更新后，价值一定不下降。

反复对两边施加 $T^{\pi'}$ 算子（迭代更新）：

$T^{\pi'}V \le (T^{\pi'})^2 V \le (T^{\pi'})^3 V \le \dots \le V^{\pi'}$

根据贝尔曼期望算子的收敛性质，无限次迭代后会收敛到新策略的真实价值 $V^{\pi'}$。

**第三步：得出终极结论**

联立所有不等式可得：

$V^\pi \le T^{\pi'}V^\pi \le V^{\pi'}$

最终严格证明：$V^{\pi'} \ge V^\pi$

无论初始的 $V$ 是否精准、是否收敛、是否存在误差，只要通过贪心生成新策略，新策略的全局真实价值一定优于或等于旧策略。

### 2\.2\.2 证明带来的颠覆性思维跃迁（研究者真实心路）

1960年前所有学者的错误认知：**V必须精准收敛到 V^π，贪心提升才有效**。

证明后彻底推翻：

1\. 收敛精准V = 可以提升策略；

2\. 未收敛粗糙V = 依然可以提升策略；

3\. **策略提升的有效性，和V是否收敛无任何绑定关系**。

### 2\.3 衍生猜想（价值迭代的雏形）

研究者进一步大胆假设：

既然「跑10次评估、跑3次评估」都能提升策略，

那我**一次评估都不跑**，直接每一步对动作取 max，能不能行？

这就是价值迭代的原始猜想。

## 第三阶段：算子建模——把两步算法数学拆解

为了验证猜想，研究者将策略迭代抽象为**两个数学算子**（核心数学工具）

### 3\.1 期望算子 T^π （代表策略评估）

$(T^\pi V)(s) = \sum_a \pi(a|s)\big[r+\gamma \sum_{s'}p(s'|s,a)V(s')\big]$

含义：固定策略，按概率加权平均，更新价值（纯评估、不优化动作）

### 3\.2 最优算子 T\* （代表评估\+提升合并）

$(T^* V)(s) = \max_a \big[r+\gamma \sum_{s'}p(s'|s,a)V(s')\big]$

含义：不绑定任何策略，直接遍历所有动作、取最大回报

### 3\.3 两种算法的算子本质对比

- **标准策略迭代**：先无限次执行 $T^\pi$（收敛）→ 再贪心更新策略

- **猜想的新算法**：直接每一步执行 $T^*$，省略所有 $T^\pi$ 加权评估

## 第四阶段：核心数学证明——为什么直接取max一定收敛？

研究者必须回答一个致命问题：**去掉加权、直接max，算法会不会乱震荡、不收敛？**

最终通过**巴拿赫压缩映射不动点定理**完成终极证明。

### 4\.1 压缩映射性质（贴现MDP γ∈\(0,1\)）完整严格证明

本节补全**最优算子 T\* 是 γ\-压缩映射**的全程推导，解决核心疑问：为什么每一步迭代误差一定会缩小、绝对不会震荡发散？

**前置定义**：无穷范数（最大误差范数）

$\Vert V_1 - V_2\Vert_\infty = \max_s |V_1(s) - V_2(s)|$

含义：衡量两个价值函数在所有状态上的**最大差值**，代表全局最大误差。

**证明目标**：

$\Vert T^*V_1 - T^*V_2\Vert_\infty \le \gamma \Vert V_1-V_2\Vert_\infty$

**第一步：展开最优算子定义**

对任意状态 $s$：

$T^*V_1(s) = \max_a \big(r(s,a) + \gamma \mathbb{E}_{s'}[V_1(s')|s,a]\big)$

$T^*V_2(s) = \max_a \big(r(s,a) + \gamma \mathbb{E}_{s'}[V_2(s')|s,a]\big)$

瞬时奖励 $r(s,a)$ 只和当前状态动作有关，与价值函数无关，做差可抵消。

**第二步：利用最大值不等式基础性质**

对任意两组函数，恒成立核心不等式：

$|\max_a f(a) - \max_a g(a)| \le \max_a |f(a)-g(a)|$

通俗解释：**两个最大值的差值，不超过所有自变量对应差值的最大值**，规避max操作带来的波动。

代入本文函数：

$|T^*V_1(s)-T^*V_2(s)| \le \max_a \gamma \cdot \big|\mathbb{E}[V_1(s')]-\mathbb{E}[V_2(s')]\big|$

**第三步：期望绝对值放缩**

根据期望基本性质：$|\mathbb{E}[X]| \le \mathbb{E}[|X|]$，可得：

$\big|\mathbb{E}[V_1(s')-V_2(s')]\big| \le \mathbb{E}\big[|V_1(s')-V_2(s')|\big]$

结合无穷范数定义：$|V_1(s')-V_2(s')| \le \Vert V_1-V_2\Vert_\infty$，该上界对所有状态 $s'$ 恒成立。

**第四步：最终放缩得到压缩性质**

$|T^*V_1(s)-T^*V_2(s)| \le \gamma \cdot \Vert V_1-V_2\Vert_\infty$

对所有状态 $s$ 取最大值：

$\Vert T^*V_1 - T^*V_2\Vert_\infty \le \gamma \Vert V_1-V_2\Vert_\infty$

**证明结论**：

因为贴现因子 $0\le\gamma<1$，最优算子 $T^*$ 是**γ\-压缩映射**，每一轮迭代，全局最大误差都会被乘以一个小于1的系数，**误差严格指数级衰减**。

### 4\.2 巴拿赫不动点定理 完整落地证明（存在性\+唯一性）

基于上面压缩映射性质，直接推导两个核心结论：**最优价值 V\* 一定存在、且唯一**。

**定理前置条件（已全部满足）**

1\. 有限MDP的价值函数空间是完备度量空间；

2\. $T^*$ 是该空间上的压缩映射。

**结论1：存在性（一定有解）**

任取初始价值函数 $V_0$，构造迭代序列：$V_{k+1}=T^*V_k$

由压缩映射性质，序列误差不断收敛：

$\Vert V_{k+1}-V_k\Vert_\infty \le \gamma^k \Vert V_1-V_0\Vert_\infty \to 0 \quad (k\to\infty)$

序列是柯西序列，完备空间中必然收敛，极限即为不动点 $V^*$，满足：

$T^*V^* = V^*$

**结论2：唯一性（只有唯一全局最优解）**

假设存在两个不动点 $V^*_1,V^*_2$，满足：

$T^*V^*_1=V^*_1,\quad T^*V^*_2=V^*_2$

代入压缩映射公式：

$\Vert V^*_1-V^*_2\Vert_\infty = \Vert T^*V^*_1-T^*V^*_2\Vert_\infty \le \gamma\Vert V^*_1-V^*_2\Vert_\infty$

因 $\gamma<1$，仅当 $\Vert V^*_1-V^*_2\Vert_\infty=0$ 等式成立，即 $V^*_1=V^*_2$。

**最终落地意义**：

不管初始V是全0、随机数、旧策略价值，**价值迭代永远收敛到同一个唯一全局最优价值函数**，不会局部最优、不会震荡、不会发散。

### 4\.3 单调性兜底完整证明（为什么迭代只会变好不会变差）

补全最优算子迭代的**单调不减\+有上界**证明，彻底打消“直接max会迭代退化”的疑虑。

**第一步：证明单步迭代单调不减**

对任意合法价值函数 $V$，任取策略 $\pi$，恒有：

$T^*V(s) = \max_a Q(s,a,V) \ge \sum_a \pi(a|s)Q(s,a,V) = T^\pi V(s)$

取上一轮迭代的隐含策略，可推导出：

$V_{k+1}=T^*V_k \ge V_k$

**迭代序列全程单调不减**：$V_0 \le V_1 \le V_2 \le \dots$

**第二步：证明序列有上界**

贴现MDP中，所有状态的长期回报存在全局上界，最优价值 $V^*$ 是序列的严格上界：

$V_k \le V^*, \quad \forall k$

**第三步：单调有界收敛定理收尾**

实数序列「单调不减\+有上界」必然收敛，结合前文压缩映射结论，序列最终收敛到唯一不动点 $V^*$。

**通俗终极兜底结论**：

价值迭代每一步更新，**所有状态的价值只会不变或变好，绝对不会变差、不会震荡、不会倒退**，数学上100%稳定收敛。

最优算子满足：

$\Vert T^*V_1 - T^*V_2\Vert_\infty \le \gamma \Vert V_1-V_2\Vert_\infty$

含义：每一次迭代，价值函数的误差都会**指数级缩小**。

### 4\.2 不动点定理结论

1. **存在唯一最优不动点 V\***：即贝尔曼最优方程的解（全局最优价值）

2. **任意初始值迭代 T\* 一定会收敛**：无论初始V是全0、随机数，反复执行 $V_{k+1}=T^*V_k$ 必然收敛到 $V^*$

### 4\.3 单调性兜底证明

$T^*V \ge T^\pi V$

最大值一定大于等于加权平均值。

迭代序列单调不减、有上界，**绝对不会退化、震荡**。

## 第五阶段：最终结论——两步彻底合并，价值迭代正式诞生

### 5\.1 研究者最终厘清的等价关系

- 策略迭代：**无限次加权评估 \+ 单次贪心提升**

- 截断策略迭代：**有限次加权评估 \+ 单次贪心提升**

- 价值迭代：**0次加权评估 \+ 每步直接贪心提升**

数学上：三者全部合法、全部单调收敛，只是收敛速度、计算冗余度不同。

### 5\.2 合并的本质逻辑（最核心总结）

策略迭代：先**按旧策略加权平均算价值**，再**选新最优动作**（两步分离）

价值迭代：直接**放弃旧策略加权**，一步max同时完成：

1. 遍历所有动作的未来回报（等价评估）

2. 直接锁定当前最优动作（等价提升）

**一步干了原来两步的事，且数学上严格收敛、无理论漏洞**。

## 第六阶段：两种算法终极差异复盘（研究者视角）

| 对比维度 | 策略迭代         | 价值迭代             |
| ---- | ------------ | ---------------- |
| 核心算子 | 期望算子 T^π（加权） | 最优算子 T\*（取max）   |
| 流程结构 | 评估、提升严格分离    | 评估提升合并为单步        |
| 策略存储 | 全程维护显式策略概率数组 | 中途无策略，收敛后导出      |
| 计算冗余 | 极高，每次提升需全表收敛 | 极低，无重复收敛计算       |
| 理论基础 | 策略改进定理（有限提升） | 压缩映射\+单调收敛（全局最优） |

## 第七阶段：后世延伸（整个RL算法树的源头）

这次「两步合并」的数学突破，直接奠定了现代强化学习的根基：

1. 价值迭代 → 衍生出 TD学习、Q\-Learning（无模型版价值迭代）

2. Q\-Learning → 衍生出 DQN、深度强化学习

3. 策略迭代 → 衍生出 MC控制、PPO、A2C 等策略梯度算法

**简单说：没有这次“两步合并”的数学突破，就没有现代DRL。**

## 全文终极一句话总结

研究者先是发现**策略提升不依赖收敛的精准价值**，打破行业误区；再通过算子建模与压缩映射证明：**完全舍弃加权评估、每一步直接取动作最大值，依然能单调收敛到全局最优**，最终将「策略评估\+策略提升」两步流程数学合并，诞生了价值迭代算法。
